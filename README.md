# The Tatarstan Bridge

## About the project

The site is designed to explore the Republic of Tatarstan and help travelers:

* phrasebook with useful phrases
* Background information for tourists
* Simple and fast interface without a backend
* Switch the light/dark theme
* Adaptive layout for mobile devices

## What i have used

* **HTML5**
* **CSS3**
* **JavaScript**
* **JSON**

## Structure of the project

```
Xviizyk.github.io/
```
